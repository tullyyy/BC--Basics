Keyboard Shortcuts

  1. How do you quickly switch between different windows on your operating system? What is the shortcut?

    2. command + tab

  2. How do you move the insertion point to the beginning of the document / to the end of the document?

    3. command + up arrow key or command + down arrow key.

  3. How do you move the insertion point to the beginning of the current line / to the end of the current line?

    4. command + left arrow key or command + right arrow key.

  4. How do you select the text between your cursor and the end of the current line?

    5. shift + left/right/up/down arrow keys.

  5. How to open settings in any app?

    6. command + comma key.

  6. How do you refresh a page?

    7. command + R.

  7. How do you refresh a page while emptying a cache? Why would you want to do this?

    8. command + R + shift. It can help pages load faster.
